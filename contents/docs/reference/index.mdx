## Introduction

Welcome to the Ixian Platform's technical reference section. This area is intended for developers, node operators, and anyone seeking an in-depth understanding of the underlying technologies that power the Ixian network.

Here you will find detailed documentation on node configuration, the complete JSON-RPC Application Programming Interfaces (APIs), and other technical specifications. This information is essential for building applications on Ixian, integrating with the network, or performing advanced node administration.

This section is organized by the core components of the Ixian ecosystem.

---

### Reference Guides

*   **[Common Reference](./reference/common)**  
    Explore the foundational APIs, data structures, and functionalities that are shared across all Ixian node types. This is the starting point for understanding wallet management, transaction creation, and basic node administration.

*   **[DLT Node Reference](./reference/dlt)**  
    Dive into the specifics of the Ixian Distributed Ledger Technology (DLT) node. This guide covers the complete configuration and the specialized JSON-RPC API for interacting with the blockchain, consensus, and mining functions.

*   **[S2 Node Reference](./reference/s2)**  
    Learn about the Ixian S2 (Streaming and Signaling) node. This guide provides the full configuration details for the peer-to-peer communication layer that enables real-time, end-to-end encrypted data exchange.

*   **[Quixi Bridge Reference](./reference/quixi)**  
    Access the high-level API for building decentralized applications. The Quixi Bridge reference details the simplified RPC methods for contact management, secure messaging, and custom application data protocols, abstracting away the complexity of the underlying S2 network.
