## Introduction

This section contains the technical specifications for the Ixian S2 streaming node. The S2 node forms a decentralized, peer-to-peer communication overlay network that provides end-to-end encrypted data streaming, presence management, and signaling services for applications.

It relies on the Ixian DLT for its security model and peer discovery but operates as a distinct layer focused on real-time communication.

### S2 Node Documentation

*   **[Configuration](./s2/config)**  
    A complete reference guide for all parameters in the `ixian.cfg` file and all available command-line arguments specific to the S2 node. This includes settings for network ports, API access, and connection parameters for the underlying DLT network.

*   **[Common RPC v0](./common/rpc_v0)**  
    S2 nodes primarily expose the shared set of common RPC methods for basic administrative and wallet functions.
