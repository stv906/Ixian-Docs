## Introduction

This section provides detailed technical specifications for the Quixi Bridge. The Quixi Bridge is a high-level application and service that runs on top of the Ixian DLT and S2 layers, exposing a powerful and user-friendly API for building decentralized applications.

It abstracts away the complexities of the underlying peer-to-peer network, offering straightforward methods for contact management, secure messaging, and custom application data exchange.

### Quixi Bridge Documentation

*   **[Configuration](./quixi/config)**  
    A complete reference guide for all parameters in the `ixian.cfg` file and all available command-line arguments for the Quixi Bridge.
    
*   **[Common RPC v0](./common/rpc_v0)**  
    Quixi nodes expose the shared set of common RPC methods for basic administrative and wallet functions.

*   **[Quixi RPC v0](./quixi/rpc_v0)**  
    A comprehensive reference for the high-level JSON-RPC methods specific to Quixi. This API is the primary interface for building applications like secure messengers and includes methods for managing contacts, sending chat messages, and exchanging custom application data.
