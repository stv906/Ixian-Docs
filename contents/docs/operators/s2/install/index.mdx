## Introduction

An Ixian S2 node forms the second layer of the Ixian platform: the decentralized streaming and data relay network. S2 nodes are responsible for managing peer-to-peer presence information, discovering other clients, and relaying end-to-end encrypted data for all clients and applications.

S2 nodes work in conjunction with the DLT network, which they use as a trust anchor for discovering other S2 nodes. This section provides detailed instructions for setting up an S2 node on supported platforms.

After completing the installation, be sure to review the [**S2 Node Configuration**](./config) guide to customize your node.
