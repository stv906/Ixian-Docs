## Introduction

Welcome to the Ixian S2 Node Operator Guide. This section provides the essential information needed to set up and run an Ixian S2 streaming node.

The S2 network is a decentralized, peer-to-peer communication layer built on top of the Ixian DLT. S2 nodes are responsible for relaying end-to-end encrypted data streams, and managing presence information for all clients and applications. An S2 node relies on a connection to the DLT network to function correctly.

By running an S2 node, you help build a robust and private communication network for users around the world.
