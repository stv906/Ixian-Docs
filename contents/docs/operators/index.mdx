## Introduction

Welcome, future Ixian node operator! This section of the documentation is your central resource for setting up, configuring, and maintaining the nodes that form the backbone of the Ixian network.

Node operators are vital to the health, decentralization, and security of the entire ecosystem. By running a node, you are not just a user, you are an active participant in the network's infrastructure. Whether you want to help secure the blockchain or power the next generation of decentralized communication, this is the place to start.

---

## Choose Your Node

The Ixian platform consists of two primary types of nodes that you can operate. Each serves a distinct but complementary purpose.

### DLT Node

The **DLT (Distributed Ledger Technology) Node** is the core of the Ixian network. It is the software that maintains the blockchain, validates transactions, maintains Ixi Names records, tracks base presences, and participates in the Ixiac Proof-of-Collaborative-Work consensus. Running a DLT node directly contributes to the security and integrity of the financial ledger.

*   **[Go to the DLT Node Operator Guide](./operators/dlt)**

### S2 Node

The **S2 (Streaming and Signaling) Node** forms the decentralized communication layer. It is responsible for relaying end-to-end encrypted data, managing user presence, and enabling real-time applications like Spixi. Running an S2 node helps build a robust and private communication network for all users.

*   **[Go to the S2 Node Operator Guide](./operators/s2)**
